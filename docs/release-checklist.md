# 三笔心情上线检查清单

## 1. 环境与构建

- [ ] `npm ci` 成功
- [ ] `npm run lint` 通过
- [ ] `npm run test:run` 通过
- [ ] `npm run build` 通过
- [ ] 预览包可正常启动：`npm run preview`

## 2. 核心体验回归

- [ ] 今日页三笔模式可编辑并保存
- [ ] 今日页自由手绘可多笔、撤销、清空、保存
- [ ] 首次使用空状态引导显示正确
- [ ] 保存反馈文案显示“今天的脸已经收好了。”
- [ ] 刷新页面后当天记录可回显

## 3. 月历回看

- [ ] 月历可切换上个月/下个月/回到本月
- [ ] 有记录日期显示真实缩略脸
- [ ] 点击某天可显示详情（脸、备注、更新时间）
- [ ] 空白日期提示文案正确

## 4. 数据管理

- [ ] 设置页统计数字正确（总记录、模式、备注、同步状态）
- [ ] 导出 JSON 可下载且可解析
- [ ] 清空本地数据有二次确认

## 5. 云同步（可选）

- [ ] `.env` 已设置 `VITE_SUPABASE_URL` 与 `VITE_SUPABASE_ANON_KEY`
- [ ] Supabase 已执行 `supabase/schema.sql`
- [ ] 邮箱登录链接可发送并成功回跳
- [ ] `安全合并同步` 正常
- [ ] `本地覆盖云端` 二次确认与执行正常
- [ ] `云端覆盖本地` 二次确认与执行正常
- [ ] 最近同步时间与策略显示正确

## 6. PWA

- [ ] 设置页安装入口状态正确（可安装/已安装）
- [ ] 安装后可从主屏打开
- [ ] 断网后可启动并查看本地记录

## 7. 发布前信息

- [ ] README 与当前功能一致
- [ ] 隐私说明明确“默认本地保存，登录后可同步云端”
- [ ] 版本号已更新（如需要）
